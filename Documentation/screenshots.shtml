Installation

How to Install

Australian Weather App is a web based application. Installation on your local Pc requires a local web hosting software due to the Php coding used for this Assignment. The software our group used is WampServer which can be downloaded at http://www.wampserver.com/en/.

Wamp Server Installation

Download and install the appropriate WampServer onto your local PC and run it. You should be able to access the Apache and MySql services on your local pc now. If you are having trouble accessing localhost on your local pc. You can edit the settings of your localhost port.

The normal port for localhost is port 80. You can use Apache services to check if your port is being used by another program on your local PC by clicking on the Wamp Manager which should be located on the Status Area or Notifications Area on your PC (Bottom Right of your screen) and click on:
  • ⇨Wamp Manager
  • ⇨Apache
  • ⇨Services
  • ⇨Test Port 80

If local host is used by another program you can change the settings on your Apache Server to use another Port such as 8080. Then on your browser u can access it by typing localhost:8080. To run the files which are located in the www subfolder in your wamp folder. Your settings can be changed by accessing the httpd.config file in Apache.
  • ⇨Wamp Manager
  • ⇨Apache
  • ⇨httpd.config

Access this file and switch all the 80's with the desired port number.

Another common error is when your browser doesn't allow access to files in the www folder. You can rectify this by opening httpd.config again and removing the # before the #soLoadModule rewrite_module modules/mod_rewrite text.

Pull files from GitHub

The next step is to download the files onto your local machine. You can either pull a request from GitHub or download the ZIP files directly.

Once you have it on your local machine, extract the files into your wamp/www/ folder. Next go to your selected browser (Chome/Mozilla/Explorer) and in your Address Bar type in localhost or localhost/index.php

App Usage

Usage of this App is fairly Simple and Straight forward. Firstly, the main HTML page has a feature which shows the weather of the closes weather station to the users current location which is very convenient for users of this app.

  • Simply select the desired State from the Main Html page and a window will appear.
  • Select the suburb which weather you want to view. You view the weather details by clicking the View Details button located on the right of the suburb. The weather chart then appears.
  • There is a Refresh button on the weather chart that refreshes the details of the selected suburb page when clicked.
  • There is Add To Favourite button on the weather chart as well that allows you to save the selected suburb into your favourites list but this feature is only allowed if you are Registered and Logged in.

  • There is also a feature that allows you to view the previous days weather and forecasts of the selected suburb. You can access this by clicking the Previous Days and Forecast buttons then after the calendar pops up, simply select the date you wish to view and the details will get automatically refreshed on the window.

Logging In
  • Firstly you will have to register your account. You do this simply by entering your desired User Id into the input box at the top right of the main html page and select the Register option from the dropdown box beside it.
  • After registering, Enter your User Id and select the Log In option from the dropdown menu

  • After Logging in. You can now add selected suburbs to your favourites list and after it is added, it will be displayed on the top left of the main HTML page.
  • You can remove a suburb from your favourite list by selecting the Delete option located beside the name of the suburb.

  • After you have added suburbs to your favourites list. It can be accessed by clicking on it, after it is clicked a drop down menu of your favourite list will be displayed. Simply click on the suburb in which you would like to view the weather and a chart of the weather will pop up.
  • This page has the same functions as the details page. It has a refresh button, which refreshes the details of the weather internally, a previous days and forecast button which displays the weather of previous and forecast weather for the selected suburb after selecting the date from the pop up calendar and also a close button which closes the window.
