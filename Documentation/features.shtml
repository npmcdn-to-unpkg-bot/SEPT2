Asddional features

1. Node.js based web spider

Please find a node js file /nodeJS_webspider/server.js

Our team has created a simple web spider using node js which generates a json file that every city names are mapped to their json urls for getting data from BOM site. The json file help other teams as well.

2. Find a nearest station from a user's location(This is based on an IP service company)

Our app shows the nearest station's weather info on the landing page. Our system gets a user's latitude and longitude and calculates the nearest station.
